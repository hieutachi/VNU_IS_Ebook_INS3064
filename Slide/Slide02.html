üì¶ Variables in PHP

What is a Variable?

Variable = Container for data

A variable can change its value during program execution.

üéí Real-life Analogy

Variables are like labeled containers:

  • Label = Variable name ($name)
  • Content = Value ("John Doe")
  • You can change the content anytime

Example

<?php $name = "John Doe"; $age = 20; $gpa = 3.5; echo "Name: $name<br>"; echo "Age: $age<br>"; echo "GPA: $gpa"; ?>

üìã Variable Naming Rules

Rule Example
Must start with $ $name
Next: letter or _ $_name, $name
Can contain numbers (not at start) $name1
Case-sensitive $Name ‚â† $name

‚úÖ Correct Examples

<?php $name = "John"; // Lowercase $userName = "john123"; // camelCase (recommended) $user_name = "john123"; // snake_case $_private = "secret"; // Starts with _ $name2 = "Jane"; // Contains number ?>

‚ùå Wrong Examples

$2name = "Error"; // Starts with number $user-name = "Error"; // Contains hyphen $user name = "Error"; // Contains space

üî¢ Data Types in PHP

Type Description Example
String Character string "Hello", 'World'

üí° PHP is dynamically typed: You don't need to declare the type. PHP automatically determines the type based on the value.

üìù String Data Type

Single Quotes vs Double Quotes

Single Quotes ' '

Literal string - doesn't parse variables

$str = 'Hello $name'; // Output: Hello $name

Double Quotes " "

Parsed string - processes variables

$name = "John"; $str = "Hello $name"; // Output: Hello John

String Concatenation

<?php $name = "John"; $greeting = "Hello, " . $name . "!"; echo $greeting; // Hello, John! ?>

Common String Functions

<?php echo strlen("Hello"); // 5 - Length echo strtoupper("hello"); // HELLO echo strtolower("HELLO"); // hello echo substr("Hello", 0, 3); // Hel echo str_replace("o", "0", "Hello"); // Hell0 ?>

üî¢ Integer & Float

Integer (Whole Numbers)

<?php $int1 = 42; $int2 = -10; $int3 = 0; var_dump($int1); // int(42) ?>

Float (Decimal Numbers)

<?php $float1 = 3.14; $float2 = -2.5; $float3 = 1.5e3; // 1500 (scientific notation) var_dump($float1); // float(3.14) ?>

Type Casting

<?php $str = "123"; $num = (int) $str; // 123 $num = intval($str); // 123 ?>

Math Functions

<?php echo abs(-5); // 5 echo round(3.7); // 4 echo ceil(3.2); // 4 echo floor(3.8); // 3 echo max(1, 5, 3); // 5 echo min(1, 5, 3); // 1 echo rand(1, 100); // Random 1-100 ?>

Array - Associative

‚öñÔ∏è Comparison Operators

Value Comparison (==) vs Type Comparison (===)

<?php $a = 5; $b = "5"; // Value comparison $a == $b; // true (5 == "5") $a != $b; // false // Value AND type comparison $a === $b; // false (int !== string) $a !== $b; // true ?>
Operator Name Example Result
== Equal (value) 5 == "5" true
=== Identical (value & type) 5 === "5" false
!= or <> Not equal 5 != "5" false
!== Not identical 5 !== "5" true
> Greater than 5 > 3 true
< Less than 5 < 3 false
>= Greater or equal 5 >= 5 true
<= Less or equal 5 <= 5 true

‚ö†Ô∏è Important: Always use === and !== for strict comparison to avoid type coercion issues!

üîÄ Logical Operators

<?php $a = true; $b = false; // AND - Both must be true $a && $b; // false $a and $b; // false // OR - At least one true $a || $b; // true $a or $b; // true // NOT - Invert !$a; // false !$b; // true // XOR - Only one true $a xor $b; // true ?>
Operator Name Description Example
&& or and AND Both must be true true && false = false
|| or or OR At least one true true || false = true
! NOT Invert value !true = false
xor XOR Only one true true xor false = true

üí° Precedence: && and || have higher precedence than and and or

‚ûï‚ûñ Increment/Decrement Operators

Pre-increment/Decrement

Increment/decrement first, then use the value

<?php $x = 5; echo ++$x; // 6 (increment x to 6, then print 6) echo --$x; // 5 (decrement x to 5, then print 5) ?>

Post-increment/Decrement

Use the value first, then increment/decrement

<?php $x = 5; echo $x++; // 5 (print 5, then increment x to 6) echo $x--; // 6 (print 6, then decrement x to 5) ?>
Operator Name Example Result
++$x Pre-increment $x = 5; echo ++$x; 6
$x++ Post-increment $x = 5; echo $x++; 5 (then $x = 6)
--$x Pre-decrement $x = 5; echo --$x; 4
$x-- Post-decrement $x = 5; echo $x--; 5 (then $x = 4)

üîÄ Control Structures: If-Else

Simple If

<?php $score = 75; if ($score >= 50) { echo "Pass"; } ?>

If-Else

<?php if ($score >= 50) { echo "Pass"; } else { echo "Fail"; } ?>

If-Elseif-Else

<?php if ($score >= 90) { echo "Excellent"; } elseif ($score >= 80) { echo "Good"; } elseif ($score >= 70) { echo "Fair"; } elseif ($score >= 50) { echo "Average"; } else { echo "Poor"; } ?>

Ternary Operator

<?php $result = ($score >= 50) ? "Pass" : "Fail"; echo $result; ?>

üîÄ Switch Statement

Use switch when comparing one variable against multiple values:

<?php $day = date("l"); // Get day of week name switch ($day) { case "Monday": echo "Monday - Start of new week!"; break; case "Tuesday": echo "Tuesday"; break; case "Wednesday": echo "Wednesday - Midweek"; break; case "Friday": echo "Friday - Almost weekend!"; break; case "Saturday": case "Sunday": echo "Weekend - Rest!"; break; default: echo "Invalid day"; } ?>

üí° Key Points:

  • Use break to exit after each case
  • Multiple cases can share the same code
  • default handles unmatched values
  • Switch is faster than multiple if-else for many conditions

üîÑ For Loop

Basic For Loop

<?php for ($i = 1; $i <= 5; $i++) { echo "Count: $i<br>"; } // Output: Count: 1, Count: 2, Count: 3, Count: 4, Count: 5 ?>

For Loop with Array

<?php $fruits = ["Apple", "Banana", "Orange"]; for ($i = 0; $i < count($fruits); $i++) { echo $fruits[$i] . "<br>"; } ?>

Nested For Loop (Multiplication Table)

<?php echo "<table border='1'>"; for ($i = 1; $i <= 10; $i++) { echo "<tr>"; for ($j = 1; $j <= 10; $j++) { echo "<td>" . ($i * $j) . "</td>"; } echo "</tr>"; } echo "</table>"; ?>

üîÑ While & Do-While Loops

While Loop

Checks condition first, then executes

<?php $count = 1; while ($count <= 5) { echo "Count: $count<br>"; $count++; } ?>

Do-While Loop

Executes at least once, then checks condition

<?php $num = 1; do { echo "Number: $num<br>"; $num++; } while ($num <= 5); ?>

While Loop

  • Checks condition first
  • May not execute if condition is false
  • Use when condition may be false initially

Do-While Loop

  • Executes at least once
  • Checks condition after execution
  • Use when you need guaranteed execution

üîÑ Foreach Loop

Foreach with Indexed Array

<?php $colors = ["Red", "Green", "Blue"]; foreach ($colors as $color) { echo "$color<br>"; } ?>

Foreach with Associative Array

<?php $student = [ "name" => "John Doe", "age" => 20, "gpa" => 3.5 ]; foreach ($student as $key => $value) { echo "$key: $value<br>"; } ?>

Foreach with Multidimensional Array

<?php $students = [ ["name" => "A", "score" => 85], ["name" => "B", "score" => 90] ]; foreach ($students as $index => $s) { echo ($index + 1) . ". {$s['name']}: {$s['score']} points<br>"; } ?>

üí° Best Practice: Use foreach for arrays - it's cleaner and safer than for loops!

‚öôÔ∏è Functions

Basic Function

<?php function sayHello() { echo "Hello World!"; } sayHello(); // Call function ?>

Function with Parameters

<?php function greet($name) { echo "Hello, $name!"; } greet("John"); // Hello, John! ?>

Function with Default Value

<?php function greetWithDefault($name = "Guest") { echo "Hello, $name!"; } greetWithDefault(); // Hello, Guest! greetWithDefault("Jane"); // Hello, Jane! ?>

Function with Return Value

<?php function add($a, $b) { return $a + $b; } $sum = add(5, 3); // 8 echo $sum; ?>

‚öôÔ∏è Functions with Type Hints (PHP 7+)

Type Hints for Parameters and Return

<?php function calculateArea(float $width, float $height): float { return $width * $height; } $area = calculateArea(5.5, 3.2); // 17.6 echo $area; ?>

Nullable Types

<?php function findUser(?int $id): ?string { if ($id === null) { return null; } return "User $id"; } echo findUser(123); // User 123 echo findUser(null); // (empty) ?>

üí° Benefits of Type Hints:

  • Better code documentation
  • IDE autocomplete support
  • Early error detection
  • Improved code reliability

üìù Example 1: Calculate Average Score

Scenario

Calculate average score and assign grade based on the average

Code

<?php $math = 8.5; $physics = 7.0; $chemistry = 9.0; // Calculate average $average = ($math + $physics + $chemistry) / 3; $average = round($average, 2); // Grade classification if ($average >= 9) { $grade = "Excellent"; } elseif ($average >= 8) { $grade = "Good"; } elseif ($average >= 7) { $grade = "Fair"; } elseif ($average >= 5) { $grade = "Average"; } else { $grade = "Poor"; } echo "Average score: $average<br>"; echo "Grade: $grade"; ?>

Explanation

  • Calculate average: sum of scores divided by count
  • Round to 2 decimal places for readability
  • Use if-elseif-else to classify grade
  • Display both average and grade
